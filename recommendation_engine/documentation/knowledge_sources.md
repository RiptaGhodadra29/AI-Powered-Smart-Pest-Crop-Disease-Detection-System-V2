# Disease Knowledge Base Sources

## Primary Sources

1. PlantVillage Dataset Documentation
2. PlantDoc Dataset Documentation
3. FAO Agricultural Disease Resources
4. Agricultural University Publications
5. Government Agriculture Extension Publications
6. Peer-Reviewed Research Papers

---

## Validation Rules

Every disease record should be validated using at least:

- 2 trusted agricultural sources

Verification Areas:

- Scientific Name
- Symptoms
- Cause
- Treatment
- Prevention
- Monitoring Actions

---

## Recommendation Principles

Recommendations must:

- Be farmer-friendly
- Be easy to understand
- Avoid excessive scientific jargon
- Include prevention guidance
- Include monitoring guidance
- Include organic options where applicable
- Include chemical options where applicable
